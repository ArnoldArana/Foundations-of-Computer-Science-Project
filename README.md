# Project Overview: Trending YouTube Videos
## Foundations of Computer Science

This project was developed with the primary objective of learning and mastering the fundamental tools of Data Science in Python. The focus is placed on the practical use of specialized libraries and an interactive development environment for data analysis.

### Key Technical Components:
* **Data Handling:** Learning how to use the `json` library to manage complex metadata and category mapping.
* **Data Manipulation:** Intensive use of `pandas` to master essential DataFrame operations, including data cleaning, filtering, transformation, and aggregation.
* **Environment:** Implementation of the entire workflow within **Jupyter Notebook** to explore code documentation and the reproducibility of the analysis.
