# Trending YouTube Videos Analysis
**Foundations of Computer Science Project**

## 📖 Project Overview
This project explores the dynamics of viral content through the lens of the **Trending YouTube Videos** dataset. Utilizing **Python** within a **Jupyter Notebook** environment, the workflow integrates data ingestion, cleaning, and structured analysis to uncover patterns in global media trends.

## 🛠️ Key Technical Components

### 🗄️ Data Handling
* **JSON Parsing:** Leveraged the `json` library to parse complex metadata and category mappings.
* **Data Integration:** Combined hierarchical JSON data with tabular datasets for comprehensive analysis.

### 🐼 Data Manipulation
* **Pandas Operations:** Utilized `pandas` for efficient DataFrame operations, including:
    * Data cleaning and type conversion.
    * Advanced filtering and conditional selection.
    * Data aggregation and grouping for trend identification.

### 📓 Environment
* **Jupyter Notebook:** Conducted end-to-end analysis within an interactive framework to ensure reproducible, modular, and well-documented results.

## 🚀 Getting Started

### Prerequisites
To run this project, you will need the following libraries:
- Python 3.x
- Pandas
- Jupyter Notebook / JupyterLab

### Usage
1. Clone the repository.
2. Ensure the dataset files are in the project directory.
3. Launch the notebook: `jupyter notebook`
